## Stack input

0. `Integer`: base.
1. `Integer`: exponent.

## Stack output

0. `Integer`: result of the exponential operation modulo 2<sup>256</sup>.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| `10` | `100` | * | `2` | `4` |
| `2` | | * | `2` | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.

## Dynamic gas

`dynamic_gas = {gasPrices|expByte} * exponent_byte_length`

The dynamic gas is simply function of how many bytes we need to represent the exponent in [binary](https://en.wikipedia.org/wiki/Binary_number).
