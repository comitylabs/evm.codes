## Stack input

0. `Integer`: 0-based byte index starting from the most significant byte.
1. `Integer`: value.

## Stack output

0. `Integer`: the indicated byte at the least significant position. If the byte index is out of range, the result is 0.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| `31` | `0xFF` | * | `30` | `0xFF` |
| `0xFF` | | * | `0xFF00` | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
