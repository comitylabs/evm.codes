## Stack input

0. `a`: first binary value.
1. `b`: second binary value.

## Stack output

0. `a | b`: the bitwise OR result.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| `0xF0` | `0xFF` | * | `0xFF` | `0xFF` |
| `0xF` | | * | `0xFF` | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
