## Stack input

0. `Integer`: value to duplicate.

## Stack output

0. `Integer`: duplicated value.
1. `Integer`: original value.

## Example

| Input | Output |
|------:|-------:|
| `1` | `1` |
| | `1` |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
- Stack overflow.
