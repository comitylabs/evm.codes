## Stack input

0. `b`: size in byte - 1 of the integer to sign extend.
1. `x`: integer value to sign extend.

## Stack output

0. `y`: integer result of the sign extend.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| `0` | `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` | * | `0` | `0x7F` |
| `0xFF` | | * | `0x7F` | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
