## Stack input

0. `Address`: 20-byte address of the contract to query.

## Stack output

0. `Integer`: byte size of the [code](/about).

## Example

| Input | Output |
|------:|-------:|
| `0x5B38Da6a701c568545dCfcB03FcB875f56beddC4` | `47` |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
