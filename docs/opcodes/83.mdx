## Stack input

0. ignored value.
1. ignored value.
2. ignored value.
3. `value`: value to duplicate.

## Stack output

0. `value`: duplicated value.
1. ignored value.
2. ignored value.
3. ignored value.
4. `value`: original value.

## Example

| Input | Output |
|------:|-------:|
| `0` | `1` |
| `0` | `0` |
| `0` | `0` |
| `1` | `0` |
| | `1` |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
- Stack overflow.
