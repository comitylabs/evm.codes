*Index 1 is top of the stack. See [PUSH](/#60).*

## Stack input

0. `a`: integer numerator
1. `b`: integer denominator.

## Stack output

0. `a // b`: integer result of the integer division. If the denominator is 0, the result will be 0.

## Examples

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `10` | `1` | * | `1` | `1` | `0` |
| `2` | `10` | | * | `2` | `2` | |


## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
