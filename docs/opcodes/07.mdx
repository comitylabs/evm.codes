## Notes

All values are treated as two’s complement signed 256-bit integers. Note the overflow semantic when −2<sup>255</sup> is negated.

## Stack input

0. `Integer`: numerator.
1. `Integer`: denominator.

## Stack output

0. `Integer`: result of the signed integer modulo. If the denominator is 0, the result will be 0.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
