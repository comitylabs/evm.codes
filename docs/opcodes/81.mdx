## Stack input

0. `a`: ignored value.
1. `b`: value to duplicate.

## Stack output

0. `b`: duplicated value.
1. `a`: ignored value.
2. `b`: original value.

## Example

| Input | Output |
|------:|-------:|
| `0` | `1` |
| `1` | `0` |
| | `1` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
- Stack overflow.
