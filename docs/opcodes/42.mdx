## Stack output

0. `timestamp`: unix timestamp of the current block.

## Example

|  \* | Input |       Output |
| --: | ----: | -----------: |
| `1` |       | `1636704767` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:

- Not enough gas.
- Stack overflow.
