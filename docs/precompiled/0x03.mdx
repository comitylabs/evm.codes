## Inputs

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| `[0; length[` | data | Data to hash with RIPEMD-160 |

## Output

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| [0; 31] (32 bytes) | hash | The result 20-byte hash right aligned to 32 bytes |

 If not enough gas was given, then there is no return data.

## Example

| Input | Output |
|------:|-------:|
| `0xFF` | `2c0c45d3ecab80fe060e5f1d7057cd2f8de5e557` |

Reproduce in playground:

    // First place the parameters in memory
    PUSH1 FF // data
    PUSH1 0
    MSTORE

    // Do the call
    PUSH1 20 // retSize
    PUSH1 20 // retOffset
    PUSH1 1 // argsSize
    PUSH1 1F // argsOffset
    PUSH1 3 // address
    PUSH4 FFFFFFFF // gas
    STATICCALL

    // Put the result alone on the stack
    POP
    PUSH1 20
    MLOAD
